Title : AI Resume Screening & Ranking System

An AI-powered Applicant Tracking System (ATS) using Transformers (Sentence-BERT) and NLP to semantically match resumes with job descriptions.

Features
- Semantic resume-job matching using BERT embeddings
- Skill gap analysis (missing skills detection)
- Project relevance scoring
- Experience-based ranking
- Hybrid AI scoring system
- Streamlit web interface
- Batch resume processing

Tech Stack :
- Python
- Sentence-BERT (Transformers)
- NLP
- Scikit-learn
- Streamlit
- PyMuPDF
- Pandas

Output :
- Ranks resumes
- Shows match percentage
- Displays missing skills
